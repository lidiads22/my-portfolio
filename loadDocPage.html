Loan Document Processing System

I designed and developed a full-stack automation platform for loan document validation and compliance reporting. Built with PHP and MySQL, deployed on AWS EC2.

Project Highlights

  • Automated compliance validation using custom PHP logic
  • Real-time report generation for document status
  • Scheduled cron jobs for nightly checks and notifications
  • Robust error logging and monitoring
  • Secure AWS EC2 deployment & credential management
  • Admin dashboard with visualized results and status tracking
  • phpMyAdmin for database management and debugging
